# Team A-F IT Services App

## Project Goal

This project is a cloud-hosted IT services web application designed to demonstrate:

- Static website hosting
- Backend API integration
- Cloud databases
- Infrastructure as Code
- CI/CD automation
- Monitoring and logging
- Secure cloud networking

---

## Team Roles

### App & Front-End Specialist
Responsible for:
- HTML/CSS/JavaScript
- Resume website
- UI/UX design

### DevOps & Backend Engineer
Responsible for:
- GitHub repository organization
- Backend planning
- CI/CD pipelines
- Cloud deployment

### Security & Network Engineer
Responsible for:
- Cloud networking
- IAM permissions
- HTTPS and DNS planning
- Security documentation

---

## Planned Cloud Architecture

Frontend:
- GitHub Pages or Azure Static Web Apps

Backend:
- Azure Functions API

Database:
- Azure CosmosDB

Security:
- Azure NSGs and IAM roles

DNS:
- Azure DNS

HTTPS:
- Azure CDN + HTTPS
